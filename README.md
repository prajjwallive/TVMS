Welcome to the Temple Visitor Management System (TVMS) GitHub repository! TVMS is a web-based application built using various technologies, including Tailwind CSS, Flowbite, SweetAlert, XAMPP Server, SQL, PHP, HTML, CSS, and JavaScript. This project aims to provide an efficient and user-friendly solution for managing temple visitors and related activities. Please note that this project uses the name "ISKCON" and "ISKCON Chitwan" as examples for demonstration purposes, and it is not intended to defame or commercialize these organizations personally.

## Features

TVMS offers a range of features to help temples manage their visitor data and operations effectively:

- **Visitor Registration:** Easily register visitors with their details, including name, contact information, and purpose of visit.

- **Visitor Count:** Keep track of the number of visitors entering the temple premises.

- **Graphical Data Representation:** Generate graphical representations of visitor data for insights and analysis.

- **Book Sales:** Manage the sale of books related to temple teachings and literature.

- **User Data Management:** Upload, search, and efficiently manage user data with a well-designed user interface.

## Getting Started

To use TVMS, follow these steps:

1. **Install XAMPP Server:** Download and install XAMPP Server on your system. You can find installation instructions [here](https://www.apachefriends.org/index.html).

2. **Download Repository:** Clone or download this GitHub repository to your local machine.

3. **Clear htdocs Content:** Remove the existing content in the `htdocs` folder of your XAMPP installation directory.

4. **Import Database:** Create a new database named "tvms" in your XAMPP phpMyAdmin. Then, import the SQL database file provided in this repository.

5. **Configure Database Connection:** Update the database connection details in the PHP files located in the project to match your local database settings.

6. **Run the Application:** Start the XAMPP Server and open your web browser. Navigate to `http://localhost/TVMS` to access the TVMS application.
